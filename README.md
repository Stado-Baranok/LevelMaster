🧠 Концепция
LevelMaster — это не просто уровень. Это точный инструмент для определения отклонения от горизонтали/вертикали:

✅ Отображение угла наклона в градусах (от -90° до +90°).

✅ Визуальная индикация пузырька уровня (графическая шкала).

✅ Цветовая индикация — зелёный (горизонт), жёлтый (небольшое отклонение), красный (сильный наклон).

✅ Калибровка — установка нулевого положения (усреднение).

✅ Звуковой сигнал при достижении уровня (опционально).

✅ Настройка чувствительности — порог срабатывания.

✅ Режим вертикали — проверка вертикальности (отвес).

✅ Сохранение калибровки в конфигурационный файл.

✅ Имитация датчика (для тестирования без аппаратного обеспечения).

🚀 Как запустить
Для работы с реальным уровнем потребуется акселерометр (в мобильных устройствах). В тестовом режиме используется симуляция с помощью клавиатуры или случайных чисел.

bash
# Python (Tkinter)
pip install tk  # обычно уже установлена
python level_python.py

# C++ (Qt)
qmake && make
./level_cpp

# Java (Swing)
javac level_java.java && java level_java

# C# (WPF)
dotnet run

# Go (консоль)
go run level_go.go

# Rust (консоль)
cargo build --release && ./target/release/level_rs

# JavaScript (Node.js)
node level_js.js

📦 Содержимое репозитория
Файл	Язык	Особенности
level_python.py	Python	Tkinter GUI, пузырьковая шкала, калибровка, звук через winsound/playsound
level_cpp.cpp	C++	Qt, QGraphicsView с анимацией пузырька, QSound
level_java.java	Java	Swing, рисование пузырька, Timer, звук через Toolkit.beep
level_cs.cs	C#	WPF, Canvas с анимацией, DispatcherTimer, MediaPlayer
level_go.go	Go	консоль, цветная шкала ASCII, звук через системный beep
level_rs.rs	Rust	консоль с termion, цветная шкала, beep через print!("\x07")
level_js.js	JavaScript	Node.js + readline, ASCII-шкала, beep через process.stdout.write
🔮 Расширенные функции
Режим вертикали — переключение между горизонтальным и вертикальным уровнем.

Звуковая обратная связь — частота звука меняется в зависимости от отклонения.

Автокалибровка — усреднение за несколько секунд.
